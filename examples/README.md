# Examples - Proyectos de Ejemplo

Este directorio contiene proyectos de ejemplo que demuestran diferentes características y patrones de uso de GOX Framework.

## Ejemplos Disponibles

- `basic/` - Aplicación básica con páginas simples
- `blog/` - Blog con autenticación y base de datos
- `ecommerce/` - E-commerce con servicios distribuidos
- `dashboard/` - Dashboard administrativo con componentes complejos
