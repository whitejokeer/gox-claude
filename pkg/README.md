# PKG - Paquetes Públicos

Este directorio contiene los paquetes públicos de GOX Framework que pueden ser utilizados por aplicaciones externas.

## Paquetes

- `parser/` - Parser de archivos .gox
- `compiler/` - Compilador .gox a Go
- `router/` - Sistema de routing
- `server/` - Servidor de desarrollo
- `config/` - Sistema de configuración
